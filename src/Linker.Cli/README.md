# Linker.Cli

The command line program for managing links.

## Feature

### Link

1. Add link
1. List links
1. Get link
1. Update link
1. Delete link
1. Search link
1. Visit link
1. Export link

### List

1. Add list
1. List lists
1. Get list
1. Update list
1. Delete list
1. Add link into list
1. Remove link from list
